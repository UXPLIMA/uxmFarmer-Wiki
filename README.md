# uxmFarmer Wiki

In this repository, you can find default configuration files of latest uxmFarmer version and also a wiki for learning everything related to uxmFarmer.

## Adding New Language Support

You can find language files inside languages folder, copy an existing language folder and translate everything inside it then create a PR including the new translated files. Our staff will check and merge your work if its suitable.

Click [here](https://github.com/UXPLIMA/uxmFarmer-Wiki/wiki) to reach the wiki.
